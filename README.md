# System Monitor (Java)

Jednoduchá Java desktopová aplikace pro sledování vytížení:
- CPU
- RAM
- Disku

## Funkce
- paralelní monitorování systémových prostředků
- grafické zobrazení historie
- ukládání dat do CSV
- konfigurovatelné pomocí config.properties
- bez externích knihoven
